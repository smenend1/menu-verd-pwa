# Menú Verd · PWA v4.0

PWA per generar menús saludables diaris i setmanals per a menors.

## Novetats principals

- Codi separat en fitxers: `index.html`, `style.css`, `recipes.js`, `app.js`, `sw.js`.
- Bloqueig de plats amb cadenat.
- Regeneració d'un dia concret, d'un dinar, d'un sopar o d'un entrant.
- Selector de temporada.
- Llista de la compra automàtica.
- Menús guardats amb nom.
- Mode compacte, mode complet i mode cuina.
- Botó per actualitzar la PWA i netejar la memòria cau.
- Editor de receptes més complet.
- Receptes amb freqüència màxima setmanal, temporada, al·lèrgens i aptitud.
- Resum visual setmanal.
- Revisió d'equilibri amb varietat de colors.

## Regles aplicades

- El peix només surt als sopars.
- Les croquetes només surten als sopars i com a màxim una vegada.
- El gaspatxo surt com a entrant de dinar.
- El tomàquet surt en amanides o gaspatxo, no en cremes.
- S'han afegit pasta de colors, arròs blanc, bistec, cireres, albercocs i mango.
- S'han retirat sèsam, pasta integral i pa integral.

## Publicació a GitHub Pages

Puja tots els fitxers a l'arrel del repositori i activa GitHub Pages a `Settings > Pages > Deploy from a branch > main / root`.

## Instal·lació al mòbil

iPhone: obre la web amb Safari i fes `Compartir > Afegir a la pantalla d'inici`.
Android: obre la web amb Chrome i tria `Instal·la l'aplicació` o `Afegeix a la pantalla d'inici`.
Si veus una versió antiga, prem `Actualitza app` dins la PWA.
